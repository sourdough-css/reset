# Sourdough CSS reset

Base styles for Sourdough. Built on top of:
[Normalize.css](https://github.com/necolas/normalize.css).

## Installation

Install [nodejs](http://nodejs.org) (comes with npm).

```
npm install sourdough-reset
```

## Build

Install dependencies:
```
npm install
```

Run the build:
```
npm run build
```

## Browser support

* Modern Browsers
* Internet Explorer 9+
