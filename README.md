# Sourdough CSS reset

Base styles for Sourdough. Built on top of:
[Normalize.css](https://github.com/necolas/normalize.css).

## Installation

Install [iojs](http://iojs.org) (comes with npm).

```
npm install sourdough-base
```

## Build

```
npm install
```

To generate a build:

```
npm run build
```

## Browser support

* Modern Browsers
* Internet Explorer 8+
